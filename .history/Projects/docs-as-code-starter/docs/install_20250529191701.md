## Requirements

Before you begin, make sure you have:

- **Python 3.7+**
- **pip**
- **git**

---

## Quick Setup

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Dom-Caracappa/docs-as-code-starter.git
    cd docs-as-code-starter
    ```

2. **Create and activate a virtual environment:**

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3. **Install the dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

4. **Start the local development server:**

    ```bash
    mkdocs serve
    ```

## Troubleshooting

Common installation issues:

**`mkdocs: command not found`**  
‚Üí You may be outside your virtual environment. Run:

```bash
source .venv/bin/activate
```

**`ModuleNotFoundError: No module named 'mkdocs'`**  
‚Üí Run `pip install -r requirements.txt` again to ensure dependencies are installed.

**Using Windows?**  
Activate your environment with:

```powershell
.venv\Scripts\activate
```
